## ✨ 功能说明

### 🎨 界面美化（8 项视觉特效）

| 特效 | 说明 | 默认 |
|------|------|------|
| 🔘 圆角卡片 | 20px 圆角，告别生硬直角 | ✅ |
| 🌫️ 悬浮阴影 | 卡片阴影 + 悬停上浮动画 | ✅ |
| 💊 胶囊按钮 | 50px 圆角按钮，现代感十足 | ✅ |
| 🧊 玻璃拟态 | 半透明背景 + 12px 高斯模糊 + 微光边框 | ❌ |
| 📏 紧凑布局 | 缩小内边距和卡片间距 | ❌ |
| 🌈 渐变标题 | 标题文字渐变色，支持自定义双色调色器 | ✅ |
| 🌟 动态悬停 | 卡片悬停上浮 4px + 加深阴影 + 边框高亮 | ✅ |
| 📜 极简滚条 | 6px 半透明滚动条，隐藏轨道 | ✅ |

### 📦 插件收纳系统（Bartender 模式）

将所有插件按钮分为三个区域，通过**拖拽**自由分配：

| 区域 | 说明 |
|------|------|
| 🌟 主页可见 | 常驻主界面，一键可达 |
| 📦 收纳箱 | 次级菜单，点击「收纳箱」按钮展开 |
| 🚫 彻底隐藏 | 不常用的插件，眼不见为净 |

### 🧠 智能交互

- **Glassmorphism 毛玻璃弹窗** — 25px 高斯模糊 + 200% 饱和度 + 30px 深度阴影
- **Tab 标签页** — 界面美化 / 收纳设置 / 关于插件 三页切换
- **智能防遮挡** — 自动拦截原生 `showModal`，其他弹窗弹出时自动收起收纳箱
- **点击即走** — 收纳箱内点击按钮后 150ms 自动关闭
- **实时渲染** — MutationObserver 监听 DOM 变化，新按钮自动美化
- **GitHub 检查更新** — 内置 API 自动检测最新版本
- **配置持久化** — 所有设置自动保存到 localStorage，刷新不丢失
- **防重复挂载** — 支持热重载，修改代码后直接生效

## 📖 使用说明

### 安装方式

#### 推荐方式：通过 UTools 插件功能导入

1. 下载 `UTools-Beautifier-V1.0.0.js`
2. 打开 UTools 页面 `http://192.168.100.1:2333/`
3. 点击页面顶部的 **「插件功能」** 按钮
4. 点击 **「增加插件」**
5. 在弹出的文件选择器中选择下载的 `.js` 文件
6. 刷新页面即可生效

#### 备用方式：控制台注入

1. 打开 UTools 页面 `http://192.168.100.1:2333/`
2. 按 `F12` 打开开发者工具
3. 切换到 **Console** 标签
4. 粘贴 `UTools-Beautifier-V1.0.0.js` 的全部内容，按 **回车**

### 使用指南

#### 🎨 界面工坊

点击页面底部的 **🎨 界面工坊** 按钮打开设置面板，包含三个标签页：

**1. 界面美化** — 勾选/取消 8 项视觉特效
- 渐变标题支持自定义颜色：点击色块选择器调整左右渐变色

**2. 收纳设置** — 拖拽路由分配
- 将插件胶囊拖到「主页可见」→ 按钮出现在主界面
- 拖到「放入收纳箱」→ 按钮移入次级菜单
- 拖到「彻底隐藏」→ 按钮完全隐藏

**3. 关于插件** — 版本信息与更新检查
- 点击「检查更新」自动连接 GitHub API 检测最新版本

#### 📦 收纳箱

点击页面底部的 **📦 收纳箱** 按钮打开次级菜单，查看被收纳的插件。点击其中的按钮后自动关闭。

## 🎯 设计哲学

> **少即是多，藏优于删。**
>
> 不是每个插件都需要常驻主界面，但每个插件都值得被保留。
> 通过「收纳」而非「移除」，让界面保持整洁的同时不丢失功能。
